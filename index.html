2. What Data We Collect

We do not require users to log in or provide personal information to use our app. However, the following types of data may be collected either directly by us or through our trusted third-party processors:

  • Device Information: Model, OS version, device language, etc.
  • Usage Data: App usage patterns, feature interactions, and time spent.
  • Crash Data: Logs and error traces in case the app crashes.
  • Identifiers: UUIDs used for analytics and diagnostics (not linked to personal identity).

Note: All data is anonymized and cannot be used to personally identify any user.

3. How We Use Your Data

The data we collect is strictly used to:

  • Improve application stability and performance
  • Enhance user experience and feature functionality
  • Conduct analytics to better understand usage patterns

We do not use your data for personalized advertising. The app is fully ad-free.

4. AccessibilityService API (Android Only)

Zen Detox Phone uses the AccessibilityService API to enable core features such as:

  • App blocking after a time or usage threshold
  • Displaying reminder screens for time-sensitive apps

This service operates only with your explicit permission and is solely used to support focus and productivity features.

We do not collect, store, or share any personal or sensitive information accessed via the AccessibilityService API.

5. Screen Time API (iOS Only)

Zen Detox Phone uses Apple's Screen Time API with your explicit permission to enable focus-related features such as:

  • Time-based app blocking
  • Launch-based app limits
  • Daily usage insights and reports

This same permission is required to generate your personalized usage reports within the app.

All app usage data and block-related information resides entirely on your device, and is stored securely by Apple. Zen Detox Phone never has access to this data, ensuring that your app activity remains 100% private and secure—strictly between you and your iPhone or iPad.

6. Third-Party Services

We use the following services to improve the app experience:

Google Firebase (Android & iOS)

  • Firebase Analytics: Usage trends and anonymized app behavior
  • Crashlytics: Crash diagnostics
  • Remote Config: Dynamic feature configurations
  • Performance Monitoring: App speed and responsiveness

Refer to Google’s Partner Policy for more information on how your data may be used by Firebase.

  • No personally identifiable data is stored or shared.
  • Usage data is retained only as long as needed to improve app functionality.
